Інструкція по встановленню бота через команду `pip install -e`:

1. Відкрийте командний рядок (термінал) і перейдіть до папки, в якій знаходиться код вашого бота.

2. Введіть наступну команду для встановлення бота за допомогою `pip`:
   ```
   pip install -e.
   ```

   Зверніть увагу на крапку `.` в кінці команди, яка показує, що потрібно встановити поточну папку як пакет.

3. `pip` встановить залежності та налаштує ваш бот для використання через команду `personal-helper`.

4. Тепер ви можете запустити бота, введіть команду `personal-helper` в командному рядку та дотримуйтесь інструкцій використання, які наведені вище.

Зауваження: Переконайтеся, що у вас встановлена актуальна версія `pip`, і ваша система має доступ до Інтернету для завантаження необхідних пакетів.

Якщо у вас виникають проблеми або вам потрібна додаткова допомога, будь ласка, повідомте мене.
